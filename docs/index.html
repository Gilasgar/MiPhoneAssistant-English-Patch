Mi Phone Assistant English Patch

A free English translation patch for Xiaomi Mi Phone Assistant 4.2.1028.10 (小米手机助手, sometimes called Mi PC Suite). The app is only available in Chinese and has no built-in language setting — this patch translates the user interface to English by replacing the app's XML UI resource files.

Download the latest release   View on GitHub

How to change Mi Phone Assistant language to English

  1. Download and extract the release zip.
  2. Double-click Run-English-Patch.bat (it asks for Administrator rights).
  3. Choose Apply English patch from the menu. A backup of the original files is created automatically.
  4. To undo, run it again and choose Restore original files from a backup.

What gets translated

Main toolbar, connect screen, backup and restore pages, flash screen with Recovery and Fastboot instructions, settings, menus, and dialogs. Images are intentionally not translated, for better compatibility and to avoid UI misalignment. The patcher verifies the installed version and only patches 4.2.1028.10.
